# 文件批量重命名工具

## 20250507 由于有诸多相同功能的程序，此项目不再维护。

## 环境依赖

- [.NET 8.0运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0)

## 功能清单

对于一个指定文件夹中的指定文件：

- 按特定模式递推编号来重命名
- 在文件名中查找并替换特定字符串
